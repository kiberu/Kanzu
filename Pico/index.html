Introducing Pico

A stupidly simple & blazing fast, flat file CMS.

Making the web easy.

DOWNLOAD NOW

Stupidly Simple

Picos makes creating and maintaining a website as simple as editing text files.

Blazing Fast

Pico is seriously light weight and does not use a database, making it super fast.

No Database

Pico is a "flat file" CMS, meaning no database woes, no MySQL queries, nothing.

Markdown Formatting

Edit your website in your favorite text editor usinf =g a simple markdown formatting.

Twig Templates

Pico uses the Twig templating engine, for powerful and flexible themes.

Open Source

Pico is completely free and open source, released under the MIT license.
